✅ 🧠 APPLICATION OVERVIEW (What we are building)

👉 This tool is used to:
	•	Read source Excel data
	•	Apply filters (CUT, SAR, SABB)
	•	Convert halalas → SAR
	•	Calculate balances
	•	Apply financial formulas
	•	Split into categories (DCC / GC / etc.)
	•	Generate final NII estimation report

👉 Final output = Excel file with 4 tables

⸻

✅ 📊 INPUT FILES

1. Source File

Contains:
	•	Product (ALPRIE)
	•	Currency (ALCYCD)
	•	Bank (ALDCG)
	•	Balance (ALMVC1 → in halalas)
	•	Customer number (ALCS)

⸻

2. Percentage File

Contains:

Field	Purpose
statReserve	Reserve deduction
vofRate	VOF calculation
dpfRate	Deposit fund
repoRate	Repo
vatRate	VAT
niiFactor	NII factor
days	No. of days
claimRate	Claim
hssRate	HSS

👉 Always take latest row

⸻

3. Category File

Contains customer mapping:
	•	TRUST
	•	DCC
	•	GC
	•	Private Funds

⸻

4. Config (optional)
	•	ALIMNA customers
	•	PIF customers

✅ ⚙️ PROCESS FLOW (STEP BY STEP)

⸻

🔹 STEP 1: Read & Clean Data

✔ Filter:
	•	ALPRIE = CUT
	•	ALCYCD = SAR
	•	ALDCG = SABB

✔ Remove:
	•	Zero balance rows

✔ Convert:

Balance = ALMVC1 / 100

✔ Sort:
	•	By ALMVC1 (ascending)

👉 Output: Clean data list

⸻

🔹 STEP 2: TABLE 1 (Customer / Total)

👉 Purpose:
Calculate balances

✔ Logic:
	•	Sum all balances → TOTAL

👉 Optional:
	•	Customer-wise SUM (ALIMNA, PIF)

⸻

🔹 STEP 3: TABLE 2 (MAIN NII CALCULATION)

👉 This is the core business logic

⸻

📌 FORMULAS

1. Eligible Average Balance

= Total Balance

2. Eligible SAR Balance

= Eligible Avg Balance

3. Stat Reserve Deduction

= Eligible Balance × statReserve

4. Eligible Balance

= Eligible Balance - Stat Reserve


⸻

5. Computed VOF

= Eligible Balance × vofRate × days / 365

6. Deposit Protection Fund

= Eligible Balance × dpfRate × days / 365

7. HSS Share

= VOF - DPF


⸻

8. Total NII

= HSS × niiFactor

9. VAT

= NII × vatRate

10. Claim from SABB

= NII + VAT


⸻

🔹 STEP 4: CATEGORY SPLIT

👉 Group balances into:
	•	TRUST
	•	DCC
	•	GC
	•	Private Funds

✔ Logic:

Category Balance = SUM(customer balances)


⸻

🔹 STEP 5: TABLE 3 (SECTOR SPLIT)

👉 Purpose: Split into business sectors

⸻

📌 FORMULAS

DCC:

= TRUST + DCC + Private Funds

GC:

= Total Balance - DCC

Total:

= DCC + GC


⸻

🔹 STEP 6: TABLE 4 (VOUCHER / FINAL BOOKING)

👉 Final accounting values

⸻

📌 FORMULAS

NII to be booked:

= Total NII

VAT:

= NII × vatRate

Claim:

= NII + VAT
