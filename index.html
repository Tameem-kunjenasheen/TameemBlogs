Title

New OMS (CSE) Connectivity with Tadawul UAT for Certification

⸻

Short Description

Firewall changes to allow connectivity from new OMS UAT servers to Tadawul UAT environment (PXCT2 and PXMR2).

⸻

Description

This change is to enable connectivity between new OMS (CSE) UAT environment and Tadawul UAT systems (PXCT2 and PXMR2) for certification.

We need to whitelist required IPs and open firewall access from new OMS UAT servers to Tadawul UAT environment (Cash and Derivative Markets).

This connectivity is required to complete OMS certification with Tadawul before production rollout.

⸻

Justification

This change is required for OMS certification with Tadawul.

Without this connectivity, testing cannot be completed and certification will not proceed. This will delay onboarding of the new OMS system.

This is a mandatory requirement from Tadawul for certification.

⸻

Implementation Plan
	1.	Review required source and destination IPs and ports – Network Team
	2.	Configure firewall rules to allow connectivity from OMS UAT servers – Network Team
	3.	Whitelist Tadawul UAT endpoints (PXCT2 and PXMR2) – Network Team
	4.	Apply firewall changes in UAT environment – Network Team
	5.	Perform connectivity testing (telnet/ping/application test) – ITO/MSS Team
	6.	Confirm connectivity with OMS application – Application Team

⸻

Business Impact

If this change is not implemented:
	•	OMS certification with Tadawul will not be completed
	•	New OMS onboarding will be delayed
	•	Business cannot proceed with trading readiness
	•	Project timelines will be impacted

During implementation:
	•	No impact to production systems
	•	No downtime required

⸻

Technical Impact

If this change is not implemented:
	•	OMS UAT servers will not connect to Tadawul systems
	•	Certification testing will fail
	•	Required FIX and TIP connectivity will not work
	•	End-to-end OMS testing will not be completed

This change is only in UAT environment. No impact to production systems.

⸻

Backout Plan
	1.	Remove firewall rules added for this change – Network Team
	2.	Remove whitelisted IPs and endpoints
	3.	Revert firewall configuration to previous state
	4.	Validate that connectivity is blocked again

⸻

Post Implementation Verification Plan
	1.	Verify connectivity from OMS UAT servers to Tadawul UAT (PXCT2 and PXMR2)
	2.	Test FIX and TIP connectivity
	3.	Validate OMS application connectivity
	4.	Confirm successful communication with Tadawul systems
	5.	Get confirmation from application/business team

⸻

Additional Details

Destination:
	•	PXMR2 – FIX (Cash & Derivative Market), TIP
	•	PXCT2 – FIX (Cash & Derivative Market), TIP
